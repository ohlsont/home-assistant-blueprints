Computation Engine
1 Policy Decision
compute_policy() — determines energy saving ON/OFF
› Sort today's 96 prices desc, pick top N slots N = minutes_to_block / 15
› Current price ≥ cutoff → blocked_now = true
› Override: price < price_ignore_below OR PV ≥ pv_ignore_above_w → skip
› Guard: must wait min_toggle_interval_min since last change
› Output: policy_on (energy saving active or not)
Parameters & Defaults
minutes_to_block240
price_ignore_below0.6
pv_ignore_above_w0.0
min_toggle_interval_min15
Output Dataclass
PolicyComputation→ target_on, blocked_now, cutoff, ignore_by_*
2 Saving Target
compute_saving_target() — target when energy saving is ON
› If outdoor ≥ warm_shutdown_outdoor → use virtual_temperature
› Else → heatpump_setpoint - saving_cold_offset_c
› Clamp to [control_min, control_max]
Parameters & Defaults
heatpump_setpoint20.0 °C
saving_cold_offset_c1.0 °C
virtual_temperature20.0 °C
warm_shutdown_outdoor8.0 °C
control_min_c10.0 °C
control_max_c26.0 °C
3 Comfort Target
compute_comfort_target() — target when energy saving is OFF
› Check comfort_satisfied: both rooms ≥ target − margin
› Cold boost: if outdoor < cold_threshold → boost = (threshold − outdoor) / slope
› Route A: not satisfied → comfort path + boost
› Route B: OK + storage cold → max(storage, comfort) + boost
› Route C: OK + storage OK → maintenance_target
› Clamp to [control_min, control_max]
Parameters & Defaults
comfort_target_c22.0 °C
comfort_to_heatpump_offset_c2.0 °C
storage_target_c24.5 °C
storage_to_heatpump_offset_c2.0 °C
maintenance_target_c20.0 °C
comfort_margin_c0.25 °C
cold_threshold1.0 °C
max_boost3.0 °C
boost_slope_c4.0 °C
4 Final Target Router
compute_final_target() — selects saving or comfort target
› If policy_on → use saving target
› If policy_off → use comfort target
› Write-deadband: only write if delta ≥ control_write_delta_c
› Clamp to [control_min, control_max]
Write Deadbands
saving_helper_write_delta_c0.05 °C
comfort_helper_write_delta_c0.05 °C
final_helper_write_delta_c0.05 °C
control_write_delta_c0.2 °C
Output Dataclass
FinalTargetComputation→ target, source
